# Notepad

A simple note-taking application built with Flutter and Hive.

## Features

- Create notes
- Edit notes
- Delete notes
- Local storage with Hive
- Confirmation dialog before deleting
- Scrollable note page
- Input validation

## Technologies

- Flutter
- Dart
- Hive
